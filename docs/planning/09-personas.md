### 1.3 심리 & 감정

#### 주요 스트레스 시나리오

**Scenario A: 임원 피드백 사이클** (주 3회 빈도)

```
1단계 — 피드백 수신: 
  "이 정도로는 부족해. 더 깊이 있어야 한다."
  
2단계 — 반추 시작 (Rumination Loop):
  - 업무 중 계속 생각남 (30분 이상 집중력 소실)
  - 야자까지 이 생각 → 수면 불안정
  - "내가 부족해", "더 열심히 해야 해" 반복
  
3단계 — 과몰입:
  - 주말도 일생각 (문서 다시 쓰기, 메일 재검토)
  - 음식맛 없음, 짜증
  
4단계 — 번아웃:
  - 월요일 회의 참석 때 기분 안 좋음
  - 실수 빈도 증가 (피드백으로 인한 악순환)

[효과]:
이 사이클에서 "5분 쉼"으로 반추 완화 가능
→ "내가 충분해" 객관성 회복 → 성과 개선
```

#### 주요 감정 상태 (일일)

| 시간대 | 감정 | 자세 | 강점 |
|--------|------|------|------|
| **08:30** | 축처음, 피로 | 앉아있기 힘듦 | 깊은 숨 = 각성 |
| **10:30** | 집중 모드 | 키보드 타이핑 중 | 잠깐 멈춤 = 재집중 |
| **13:00** | 혼란, 아이디어 많음 | 회의실 방금 나옴 | 머리 정리 = 우선순위 명확화 |
| **15:30** | 피로, 의욕 저하 | 화면 멀어짐 (눈 피로) | 진정 = 재에너지 |
| **19:00** | 긴장 해제 + 구역감 | 의자에서 비틀거림 | 경계 해제 = 다음 일 준비 |
| **22:00** | 깊은 피로, 근심 | 침대/소파 | 명상적 정리 = 숙면 |

### 1.4 Work-Life의 교집합

#### 현재 "쉠"이라고 생각하는 것

- ✅ **유튜브 쇼츠**: 3-5분 (하지만 자신도 "이건 쉬는 게 아니야"라고 자각)
- ✅ **커피 마시기**: 7-10분
- ✅ **화장실**: 2-3분 (가장 자주)
- ✅ **동료 잡담**: 5분 정도
- ❌ **폰 게임**: "쉬는 게 아니라 더 피로"
- ❌ **SNS (카톡, 인스타)**: "볼수록 조급함"

#### 현재 "쉬지 못하는" 이유

1. **신호 부재**: "지금이 쉴 시간이다"는 명확한 marker 없음
   - 마음이 가는 대로 → 보통 쇼츠 (도파민 루프)
   
2. **죄책감**: "쉬는 동안도 이메일 왔을 텐데..."
   - 5분이 아까움 (일의 미완성 불안)
   
3. **깊이 부족**: "뭔가 했다는 느낌이 안 남"
   - 쇼츠 본 후 20분 뿌연 상태 → 더 피로
   
4. **측정 불가**: "이게 정말 효과 있나?"
   - 객관적 신호 부족 → 동기 저하

### 1.5 현재 삶의 불만족


```
"복잡한 프로젝트를 진행할 때,

```

**이 일을 못하고 있는 상태**:
- 아이디어 회의 중 "생각이 안 남"

- 쇼츠 본 후 "뭔가 생각나려다 말음"

### 1.6 전환 모멘트 (Aha Moment)

#### 언제 MongCount를 찾을까?

1. **피드백 직후** (가장 유력):
   - "내가 지금 반추 중이구나" 자각
   - 구글: "반추 생각 없애기", "잡생각 정리"
   - MongCount 검색 → "쉬면서 아이디어 잡기" 키워드 히트

2. **주말에 피로 보일 때**:
   - 파트너/친구: "넌 너무 일만 해, 쉬어야 해"
   - "쉬는 방법을 배워보자" 심정으로 검색

3. **생산성 팟캐스트/유튜브**:
   - "쉬는 게 왜 중요한가?" 콘텐츠
   - "DMN 활성화"에 대해 배우면서 → MongCount 발견

#### 첫 구매 심리

**약한 신호** (가입만 함):
- "한 번 써보자" 마음
- Free tier 충분해 보임

**강한 신호** (유료 전환):
- 2주 쓴 후 "정말 머리가 맑아지는 느낌 있다"
- 7일 아카이브 꽉 참 → "더 오래 저장하고 싶다"
- 아이디어 한 줄 5개 이상 쌓임 → "이거 잃어버리면 안 되지"

### 1.7 성공 척도 (1년 뒤)

#### 행동 변화

**Before**:
- 일일 쉼: 0-1회 (불의도적)
- 쇼츠: 하루 30분
- 수면: 5-6시간 (피로함)
- 주중 반추 사이클: 주 3회

**After** (1년 뒤 목표):
- 일일 쉼: 6회 중 3회 이상 (의도적)
- 쇼츠: 하루 5분 (필터링)
- 수면: 6-7시간 (안정적)
- 주중 반추 사이클: 주 1회 (관리 가능)

#### 내적 변화

**Before**:
- "나 부족해"
- 피드백 = 공격
- 머리 = 항상 어지러움

**After**:
- "나 충분해, 그래도 나아질 수 있지"
- 피드백 = 정보
- 머리 = 대부분 맑음

#### 실제 성과 신호

- 분기별 360도 평가에서 "차분하고 객관적" 피드백 증가
- 회의에서 질문 + 제안 빈도 증가 (집중력 회복)
- 야근 빈도 감소 (효율 개선)

### 1.8 시간대별 심리 맵 & 마이크로카피

```
┌─────────────────────────────────────────────────────┐
│ MongCount Micro-moment Strategy (P1 기준)           │
└─────────────────────────────────────────────────────┘

[08:30 Morning Slot]
상황: 출근 준비, 축처음, 아직 뇌가 깨어나는 중
심리: "오늘도 또 회의고..."
목적: 각성 + 긍정적 마음가짐
마이크로카피: "5분 숨 고르고 가요"
  → "숨 고른다" = 신체감각 (정신을 깨움)
  → "가요" = 함께하는 느낌 (동료 의식)
  
실제 행동: 침대에서 눈 뜨고 앉아서 심호흡

───────────────────────────────────────────────────────

[10:30 Mid-Morning Slot]
상황: 오전 업무 집중 중, 1-2개 이메일/회의 사이
심리: "이 정도면 점심까지 갈 수 있겠는데..."
목적: 집중력 연장 + 정신 리셋
마이크로카피: "10분이면 충분해요"
  → "10분" = "실제로 시간 있어" (죄책감 제거)
  → "충분해요" = 허락 (이 정도면 된다)
  
실제 행동: 탭을 닫고 눈 감고 앉아 있음 (or 창밖)

───────────────────────────────────────────────────────

[13:00 Post-Lunch Slot]
상황: 회의 3개 연속 종료 직후, 머리 혼란, 아이디어 많음
심리: "회의에서 뭐했더라... 어? 근데 아이디어는 좋은데..."
목적: 정보 정렬 + 우선순위 명확화 + 아이디어 포착
마이크로카피: "점심 후 한숨 돌려요"
  → "한숨" = 긴장 이완 (회의의 긴장 해제)
  → "돌려요" = 재정렬 (다음 준비)
  
실제 행동: 카페 가서 앉아 있기 (주변 소음 OK)
아이디어 한 줄 기록: "팀 권한 위임 구조 검토 필요" 등

───────────────────────────────────────────────────────

[15:30 Afternoon Slot]
상황: 오후 업무, 2-3시간 후 피로, 화면 계속 봄
심리: "커피 마셔야 되나... 근데 또 잠이 안 올까..."
목적: 에너지 회복 (자연스러운, 커피 없이)
마이크로카피: "오후 에너지 충전"
  → "에너지" = 활력 (수면 없는 회복)
  → "충전" = 배터리 리셋 (기술감)
  
실제 행동: 화면 끄고 눈 감기 (4-5분), 물 마시기
몸의 변화: 척추 스트레칭, 목 풀기

───────────────────────────────────────────────────────

[19:00 Post-Work Slot]
상황: 퇴근, 마인드 전환 필요, 가사/개인시간 시작 전
심리: "이제 집인데... 일 생각이 자꾸만"
목적: 일과 개인의 경계 설정 + 다음 활동 정신 준비
마이크로카피: "퇴근 후 마음 정리"
  → "마음" = 감정 (일의 감정 분리)
  → "정리" = 정돈 (다음을 위한 준비)
  
실제 행동: 복도 산책 또는 카페 5분
효과: "이제 집이고 나는 여기에 있다" 체감

───────────────────────────────────────────────────────

[22:00 Night Slot]
상황: 침대/소파, 침전 시간, 내일 대비 + 오늘 정리
심리: "오늘 하루 뭐했더라... 뭔가 안 된 게 있나..."
목적: 반추 완화 + 자존감 회복 + 숙면 준비
마이크로카피: "오늘 하루 천천히 돌아봐요"
  → "천천히" = 반추 아님 (명상적 정리)
  → "돌아봐요" = 함께 (혼자가 아님)
  
UI: 다크모드 자동 활성화, 밝기 감쇠
실제 행동: 침대에서 세션 (타이머 음소거, 진동만)
아이디어 한 줄: "내일은 피드백 반영해서 문서 다시 쓸 것" (기록하고 잊음)

└─────────────────────────────────────────────────────┘
```

## 2. Personas NOT in MVP (명시적 제외)

### Excluded Persona 1: 대학생/수험생

**프로필**:
- 나이: 18-25세
- 상황: 수시/정시 대비, 시험 공부
- 스케줄: 고정적이지 않음 (공부량 따라 변함)
- 현재 쉼: 스마트폰 게임, 유튜브

**제외 이유**:
- ❌ 고정 6회 알림이 맞지 않음 (자유로운 스케줄)
- ❌ "일의 성과"라는 개념이 다름 (점수/등급)
- ❌ MVP 2.5개월 내 설계 불가능
- ❌ 우선순위: P1(직장인) >> 학생

**v2+ 확장 전략**:
- Phase D (6개월+): "커스텀 시간대" 기능 추가 → 학생 맞춤
- 예: "2시간마다 쉼" 패턴 지원

### Excluded Persona 2: 학부모

**프로필**:
- 나이: 35-55세
- 상황: 직장 + 육아 병행
- 스케줄: 가정과 일의 경계 모호
- 현재 쉼: 화장실, 카페 (단편적)

**제외 이유**:
- ❌ "쉼의 정의"가 다름 (자기시간 부족)
- ❌ 6회 알림 중 3-4회는 가사/육아로 불가능
- ❌ 심리 구조 다름 (반추 아님, 책임감)
- ❌ MVP 스코프 확대 위험

**v2+ 확장 전략**:
- Phase D: "5분도 소중한 시간" 캠페인 (학부모 맞춤)
- 예: "아이 재운 후 5분 쉼" 슬롯 추가

### Excluded Persona 3: 프리랜서/자영업자

**프로필**:
- 나이: 25-45세 (주로 자영업)
- 상황: 정해진 스케줄 전혀 없음
- 수입: 불규칙
- 현재 쉼: 거의 없음 (일과 쉼의 경계 실종)

**제외 이유**:
- ❌ 6회 고정 알림이 전혀 작동 안 함
- ❌ "일의 성과 = 매출"이므로 쉼을 죄책감으로 봄
- ❌ 심리 개입 필요 (정신과 수준)
- ❌ MVP 이후 비즈니스 모델도 달라짐

**v2+ 확장 전략**:
- Phase E (1년+): "유연한 쉼" 모드 + 자영업 재무 관리 통합
- 예: "매출 순환 패턴에 따른 쉼 권고"

## 4. P1의 구체적 사례 연구

### Case Study: 실제 feedback loop

```
[Day 1 — 새벽 02:00 심야]
상황: 큰 프로젝트 피드백 받음 (임원)
감정: 충격, 좌절, 자책
반응: 문서 전체 다시 쓰기 시작

[Day 2 — 아침]
상황: 피로, 밤샘, 정신없음
감정: 분노, 무기력
행동: 회의 중 말 없음, 메일도 늦음

[Day 2 — 오후 15:30 (MongCount 알림)]
알림: "오후 에너지 충전"
선택 1: 무시 (과거) → 피로 누적 → 실수
선택 2: 참석 (MongCount) → 5분 쉼 → ???

[실제 5분 쉼 후]
현상: "아... 내가 너무 극단적으로 생각했네"
재평가: "이 정도는 할 수 있는 수정 사항이겠는데?"
재작성: 효율적인 수정 (3시간 vs 전체 재작성)
결과: 다음날 회의에서 "좋은 방향" 피드백

[Day 3 — 아침 MongCount 기록]
집중도: ☀︎☀︎☀︎ (4/5)
한 줄: "극단적 생각 → 객관적 수정으로 전환"
심리: "어제 5분 쉬길 잘했다. 이제 정상이다."

[효과 누적]
주 3회 이런 피드백 사이클에서
→ 2회 MongCount 참석으로
→ 월 1회 번아웃 완화 → 3개월 후 근본적 심리 변화
```

## 5. P1 설문 데이터 (가상)

### "당신의 쉼은?"

```
Q1. 현재 하루 중 의도된 쉼 시간은?
A1. 0분 (34%), 5분 (42%), 10분 (20%), 30분+ (4%)
   → 평균 5분 미만

Q2. 쉬는 것에 대해 느끼는 감정은?
A2. 죄책감 (68%), 불안감 (45%), 필요성 (72%), 즐거움 (28%)
   → 대부분 죄책감과 함께

Q3. 쉼이 가장 필요한 순간은?
A3. 피드백 직후 (73%), 회의 연속 후 (81%), 실수 후 (68%)
   → "위기 관리" 타입

Q4. 현재 쇼츠/게임/SNS에 쓰는 시간?
A4. 하루 15-60분 (평균 30분)
   → "쉼이라고 착각"

Q5. 쉬면 실제로 효과를 느끼나?
A5. 느낀다 (72%), 느끼지 못한다 (28%)
   → "맑아진다"는 신호가 있음 (core value)
```
