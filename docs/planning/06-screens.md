## 2. 상세 화면 설계

### S1. 홈 / 대시보드

**목적**: 사용자가 앱 진입 시 오늘의 쉼 진행도를 즉시 파악하고, 언제든 쉼을 시작할 수 있는 허브

**진입 경로**:
- 앱 실행 → 온보딩 완료 후 (또는 이미 사용자일 경우)
- 알림 → 쉼 완료 → S3 저장 후 돌아옴
- 설정/아카이브/리포트에서 홈 탭 선택

**이탈 경로**:
- [지금 3분 쉬기] → S2 (타이머 선택 가능)
- [아카이브] 탭 → S5
- [리포트] 탭 → S6
- [설정] 탭 → S9
- 알림 수신 → OS 알림 → S4 (또는 직접 S2)

**주요 컴포넌트**:
```
┌─────────────────────────────────┐
│ MongCount    [설정]             │
├─────────────────────────────────┤
│                                 │
│ 안녕하세요, [이름]              │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 오늘 3/6 ☀︎☀︎☀︎─────      │ │
│ │ 남은 시간: 19:00 (2시간 뒤) │ │
│ └─────────────────────────────┘ │
│                                 │
│ ┌─────────────────────────────┐ │
│ │                             │ │
│ │    [지금 3분 쉬기]          │ │
│ │                             │ │
│ │  대형 버튼 (100px 높이)      │ │
│ │  배경: primary color         │ │
│ │                             │ │
│ └─────────────────────────────┘ │
│                                 │
│ 마지막 쉼: 10분 전 ☀︎☀︎☀︎    │
│                                 │
├─────────────────────────────────┤
│ [홈] [아카이브] [리포트] [설정]  │
│  ●                             │
└─────────────────────────────────┘
```

**States**:
- **정상**: 위 구성
- **첫 실행**: "오늘의 첫 쉼을 시작해보세요" (텍스트만)
- **모든 알림 완료**: "오늘 쉼을 모두 완료했어요 ✨" + 진행도 만도 표시
- **로그인 미수행**: 로그인 배너 (상단, dismissible)

**기술 상세**:
- **데이터 소스**: Zustand store `restSessions[]` (Supabase Realtime 구독)
- **실시간 업데이트**: S3 저장 후 즉시 진행도 갱신 (WebSocket)
- **다크모드**: 자동 22시 전환 (background: #0F0F0F)

**인터랙션**:
- [지금 3분 쉬기] 탭 → S2로 fade transition (300ms)
- 진행도 hover → "오늘 하루를 천천히 돌아봐요" 토스트

**접근성**:
- 버튼 포커스: outline-2 offset-2
- 키보드: Tab으로 탭 전환, Enter로 버튼 활성화
- 스크린리더: "오늘 3회 완료, 6회 목표" (ARIA label)

### S2. 쉼 세션 타이머

**목적**: 5분간 집중을 방해하지 않는 순수 쉼 경험 제공. 극도로 심플한 UI.

**진입 경로**:
- S1 [지금 3분 쉬기] → 타이머 선택 팝업 → S2
- 알림 [지금 쉬기] → 바로 S2 (마지막 선택 기억)
- S1 [아카이브] 세션 클릭 → S2 재재생 (읽기 전용)

**이탈 경로**:
- 타이머 완료 → S3 자동 이동
- [쉼 중단] 버튼 → S3 (집중도 1-2 자동 제안)

**주요 컴포넌트**:
```
┌─────────────────────────────────┐
│                                 │
│                                 │
│                                 │
│            05:00                │
│           (큰 글씨)             │
│                                 │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 생각 떠오르면 잊지 말고     │ │
│ │ (띠 텍스트, 작은 글씨)      │ │
│ └─────────────────────────────┘ │
│                                 │
│            [═════════════] (프로그레스 바)
│                                 │
│        [쉼 중단]                │
│                                 │
└─────────────────────────────────┘
```

**States**:
- **정상**: 타이머 진행 (MM:SS, 초 단위 업데이트)
- **일시정지**: "⏸" 표시 + "계속?" 버튼 (전화 수신 시)
- **완료**: 자동 fade-out + S3로 transition

**기술 상세**:
- **타이머 정밀도**: requestAnimationFrame (60fps) + Service Worker 보조
- **백그라운드 진행**: Service Worker에서 타이머 계속 계산
- **크래시 복구**: localStorage에 `sessionStartTime` 저장 → 복구 시 남은 시간 재계산
- **음소거**: 기본값 ON (사용자가 조용히 쉴 수 있도록)

**인터랙션**:
- 타이머는 자동 카운트, 수동 조작 불가 (집중력 방해 방지)
- [쉼 중단]: 누르면 "정말 중단하시겠습니까?" 확인 다이얼로그
- 세션 중 다른 앱 전환: 타이머 백그라운드 계속, 화면 꺼짐 OK

**접근성**:
- 스크린리더: 타이머 숫자 실시간 읽음 (aria-live="polite")
- 색상 대비: 다크모드 배경에서 숫자 충분한 명도
- 포커스: [쉼 중단] 버튼만 (다른 컨트롤 없음)

### S3. 집중도 + 한줄 기록

**목적**: 쉼 종료 후 사용자의 "머리 맑기"를 정량화하고, 아이디어를 빠르게 캡처

**진입 경로**:
- S2 타이머 완료 → 자동 이동
- S2 [쉼 중단] → 집중도 1-2 자동 제안 (확인 필요)

**이탈 경로**:
- [저장] → S1로 복귀 (2초 후 자동)
- [건너뛰기] → S1로 복귀 (즉시)
- 자동 저장 활성 → 필드 비움 + 3초 경과 → S1 자동 이동

**주요 컴포넌트**:
```
┌─────────────────────────────────┐
│                                 │
│    5분을 마치셨어요 ✓          │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 머리가 얼마나 맑나요?       │ │
│ │                             │ │
│ │ ☁︎ 1   2   3   4   5 ☀︎    │ │
│ │ (슬라이더, 이모지 표시)     │ │
│ │                             │ │
│ │ "뿌연" ←→ "맑은"           │ │
│ └─────────────────────────────┘ │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 한 줄로 남길 말씀 있으신가요? │
│ │                             │ │
│ │ [__________ ] (자동 포커스) │ │
│ │ 0/280 자                   │ │
│ │                             │ │
│ └─────────────────────────────┘ │
│                                 │
│     [저장]      [건너뛰기]      │
│                                 │
└─────────────────────────────────┘
```

**States**:
- **정상**: 슬라이더 (기본값 3) + 텍스트 필드 (포커스)
- **중단 제안**: 슬라이더 기본값 1-2 (사용자 확인 필요)
- **로딩**: 저장 중 [저장] 버튼 → disabled 상태

**기술 상세**:
- **슬라이더**: shadcn/ui Slider 컴포넌트 (0-5 범위, step=1)
- **이모지 표시**: JavaScript 함수로 선택값에 따라 `☁︎→☀︎` 동적 변경
- **텍스트 저장**: Dexie.js debounce (500ms)
- **Supabase 동기화**: 온라인 시 비동기 POST to `idea_records`

**인터랙션**:
- 슬라이더 드래그: 실시간 이모지 변화
- 텍스트 입력: 자동 저장 (localStorage 우선)
- [저장]: POST + 성공 토스트 "저장했어요 ✓" (2초) → 자동 S1로 이동

**접근성**:
- 슬라이더: 화살표 키로 조작 가능
- 레이블: "머리가 얼마나 맑나요?" (각 라디오그룹 이름)
- 텍스트 필드: placeholder 명확

### S4. 푸시 알림 (OS 레벨)

**목적**: 사용자를 의도적으로 쉼으로 유도. 6회 정해진 시간에만 발송.

**발송 시간** (고정, 변경 불가 MVP):
- 08:30: "5분 숨 고르고 가요"
- 10:30: "한 번 멈춰서 정리할까요"
- 13:00: "점심 후 몸과 마음 리셋"
- 15:30: "오후 경항 여기까지, 깊게 쉬어봐요"
- 19:00: "일의 끝, 쉼의 시작"
- 22:00: "오늘 하루 천천히 돌아봐요"

**알림 형식** (Android Chrome):
```
┌────────────────────────────────┐
│ MongCount                      │
│                                │
│ "5분 숨 고르고 가요"            │
│                                │
│ [지금 쉬기]    [15분 뒤]        │
│ [닫기]                         │
└────────────────────────────────┘
```

**알림 형식** (iOS Safari, 홈화면 추가):
```
┐────────────────────────────────┐
│ 쉼일지                          │
│ "5분 숨 고르고 가요"            │
│                                │
│ [자세히]           [닫기]       │
└────────────────────────────────┘
```

**액션 버튼**:
- **[지금 쉬기]**: 알림 → S2 직행 (타이머 5분 기본)
- **[15분 뒤]**: 알림 → 15분 후 재알림 (최대 3회)
- **[닫기]** 또는 무시: 조용히 닫음 (강박 없음)

**기술 상세**:
- **Android**: FCM + Web Push API
- **iOS**: APNS (홈화면 추가 필수)
- **폴백**: App 내 Notification API (로컬 타이머)
- **스누즈**: `notificationSubscription.snoozeCount` 테이블에 기록

**Soft Streak 계산**:
- 하루에 3회 이상 완료 → Streak 유지
- 2회 이하 → Streak 초기화
- S1에서 표시

**특별 사항**:
- 23:00 이후: 자동 스누즈 무시 (야간 수면 방해 방지)
- 22:00 알림: 다크모드 + 밝기 감쇠 (Blue Light 차단)

### S5. 아카이브 리스트

**목적**: 최근 7일 쉼 기록을 조회하고, 7일 초과는 유료 기능으로 유도

**진입 경로**:
- S1 [아카이브] 탭

**이탈 경로**:
- 세션 카드 클릭 → S2 상세보기 (읽기 전용)
- [홈] 탭 → S1

**주요 컴포넌트**:
```
┌─────────────────────────────────┐
│ 아카이브                 [검색]  │
├─────────────────────────────────┤
│                                 │
│ 어제 (4/15)                     │
│ ┌─────────────────────────────┐ │
│ │ 10:30  ☀︎☀︎☀︎  "프로젝트..." │ │
│ │ 타이머: 5분 29초              │ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ 15:30  ☀︎☀︎☀︎☀︎ "생각 정리" │ │
│ │ 타이머: 5분 2초               │ │
│ └─────────────────────────────┘ │
│                                 │
│ 2일 전 (4/14)                   │
│ ┌─────────────────────────────┐ │
│ │ 08:30  ☁︎☁︎☁︎   "..."      │ │
│ │ 타이머: 3분 1초               │ │
│ └─────────────────────────────┘ │
│                                 │
│ ... (최대 7일까지)              │
│                                 │
│ [프리미엄으로 무제한 저장]       │
│                                 │
├─────────────────────────────────┤
│ [홈] [아카이브] [리포트] [설정]  │
│      ●                          │
└─────────────────────────────────┘
```

**States**:
- **정상**: 기록 7일치 표시
- **빈 상태** (S11): "아직 기록이 없어요. 첫 쉼을 시작해보세요."
- **7일 경계**: 8일 전 기록 접근 시 Paywall (S10)

**필터 & 검색**:
- 상단 [검색] 아이콘: 텍스트 검색 (한줄 기록)
- 필터 아이콘: 
  - 시간대 (아침/오후/저녁)
  - 집중도 (☀︎☀︎☀︎ 이상)

**기술 상세**:
- **데이터 소스**: IndexedDB (Dexie) 기본, 로그인 시 Supabase
- **렌더링**: 가상 스크롤 (500+ 기록 시 성능 최적화)
- **검색**: 클라이언트 사이드 필터링

**인터랙션**:
- 세션 카드 클릭: 상세보기 (읽기 전용, 타이머 재생 불가)
- [프리미엠으로...]: S10 Paywall로 이동

**접근성**:
- 키보드: 화살표 키로 목록 스크롤
- 스크린리더: "어제 10:30, 맑음 레벨 3, 내용: 프로젝트..."

### S6. 주간 리포트

**목적**: P1이 주간 쉼 패턴을 시각화하여 "더 잘 쉬는 사람"으로의 성장 자각

**진입 경로**:
- S1 [리포트] 탭

**이탈 경로**:
- [월간] 탭 → 월간 분석 (Paid P5, Paywall S10)
- [홈] 탭 → S1

**주요 컴포넌트**:
```
┌─────────────────────────────────┐
│ 주간 리포트           [월간]     │
├─────────────────────────────────┤
│                                 │
│ 4월 14~20일 (이번 주)           │
│                                 │
│ 월 화 수 목 금 토 일             │
│ ●  ●  ◐  ◌  ─  ●  ●            │
│ (도트: 완료/부분/미완/예정)      │
│                                 │
│ 평균 맑기: ☀︎☀︎☀︎.5          │
│                                 │
│ 가장 맑았을 때: 목요일 15:30    │
│ ☀︎☀︎☀︎☀︎☀︎ (최고 기록)      │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ AI 코멘트                   │ │
│ │ "꾸준함이 빛나는 주였어요.  │ │
│ │ 한 번의 부분 완료도 대단해요."│ │
│ └─────────────────────────────┘ │
│                                 │
├─────────────────────────────────┤
│ [홈] [아카이브] [리포트] [설정]  │
│              ●                  │
└─────────────────────────────────┘
```

**도트 해석**:
- **●**: 완료 (Soft Streak 기준 3회 이상)
- **◐**: 부분 (1-2회 완료)
- **◌**: 미완 (0회)
- **─**: 예정 (미래 날짜, 아직 온라인 싱크 안 됨)

**AI 코멘트** (서버 생성, Supabase Function):
- 긍정적: "꾸준함이 빛나는 주였어요" (5일 이상 완료)
- 격려: "월요일부터는 힘들 수 있죠" (저조한 주)
- 중립: "나만의 리듬을 찾아가고 있어요" (일관성 있음)

**기술 상세**:
- **집계**: Supabase SQL 쿼리 (주 단위 GROUP BY)
- **AI 생성**: GPT-3.5 Turbo (한국어, 최대 100자)
- **캐싱**: 매일 자정에 한 번 생성, 변경 불가

**[월간] 탭** (Paywall, S10 경유):
- 4주 비교 (4월, 3월, 2월, 1월)
