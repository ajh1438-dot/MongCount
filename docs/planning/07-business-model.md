## 2. Freemium 구조

### Free Tier (제한 없음, 7일 무료 체험)

| 기능 | 포함 | 비고 |
|------|------|------|
| **F1 강제 쉼 알림** | ✅ | 6회 고정 시간 푸시 |
| **F2 쉼 세션 타이머** | ✅ | 3/5/10분 선택, 무제한 |
| **F3 아이디어 한 줄** | ✅ | 단일 텍스트 필드 |
| **S1 홈 대시보드** | ✅ | "오늘 n/6" + "지금 3분 쉬기" |
| **S2 주간 리포트** | ✅ | 요일 도트(●◐◌─) + 평균 맑기 + AI 코멘트 (간단판) |
| **S3 다크모드** | ✅ | 자동 22시 전환 |
| **S4 아카이브** | ✅ | **7일 초과분 차단** (Paywall 게이트) |
| **음성 녹음** | ❌ | Paid P2 |
| **CSV/Notion 내보내기** | ❌ | Paid P3 |
| **멀티 디바이스 동기화** | ❌ | Paid P4 |
| **월간 리포트 + 그래프** | ❌ | Paid P5 |
| **자동 백업** | ✅ | 단일 기기 (IndexedDB + localStorage) |

### Paid Tier (₩2,900/월, 7일 무료 체험)

| ID | 기능 | 가격 | 포함 내용 |
|----|------|------|---------|
| **P1** | 무제한 아카이브 | ₩2,900 | 7일 초과분 해제 |
| **P2** | 음성 녹음 / 사진 첨부 | " | Supabase Storage (회원당 1GB) |
| **P3** | CSV / Notion 내보내기 | " | 월 1회 전체 데이터 내보내기 |
| **P4** | 멀티 디바이스 동기화 | " | 최대 3대 기기 (휴대폰+태블릿+웹) 실시간 동기화 via Supabase Realtime |
| **P5** | 월간 리포트 + 추세 그래프 | " | 월 비교 그래프 + 심화 분석 (예: "이번 달 오전 알림이 30% 더 효과적") |

### 근거

**Free는 왜 이 수준?**
- ☁︎~☀︎ 피드백 루프: 하루 6회 알림 중 최소 1회 이상 의도된 쉼 경험 → 가치 직접 체감
- 7일 아카이브: "쉼을 기록한다"는 경험 제공하되, 데이터 축적되면 유료 업그레이드 자연스러움
- 주간 리포트: Paid와 차별화하되, Free 유저도 "더 잘 쉬는 사람" 성장감 제공
- **의도**: 생각할 시간 주지 말고, 즉시 가치 실감 후 자연스러운 Paywall 진입

**Paid는 왜 이 5개?**
1. **P1 (무제한 아카이브)**: "내 기록이 사라지면 안 된다" 심리 → Core Paywall
2. **P2 (음성/사진)**: "더 풍부한 기록" → 창의성 부산물 심화
3. **P3 (내보내기)**: "데이터 자유" + 노션 권력 사용자 타겟
4. **P4 (멀티 디바이스)**: "일과 휴식을 어디서나" → 진정한 유목민 유저
5. **P5 (월간 리포트)**: "더 잘 쉬는 사람" 코칭 심화 → 장기 리텐션

## 3. Pricing Strategy

### 초기 가격: ₩2,900/월

**근거**:
- **심리학**: 월 ₩2,900 = 일일 ₩96 = "커피 한 잔"보다 싼 수준. 진입 장벽 낮음
- **경쟁 분석**: Calm 평균 ₩9,900/월 (프리미엄), Wander120 무료 (포기 가능). 중간값 설정
- **가치 전달**: 1년 후 "하루 30분 의도된 쉼"으로 월 ₩2,900 = 정당함
- **사이드 프로젝트**: 유료전환 4% × 10,000 MAU = 월 ₩116만 (사이드 가능한 수준)

### 6개월 후: ₩4,900/월 인상 검토

**조건**:
- D30 리텐션 40% 이상 달성
- App Store 평점 4.5/5 이상 유지
- MAU 10,000 확보

**전략**:
- **기존 사용자 Grandfathering**: ₩2,900 요금 유지 (평생)
- **신규 사용자**: ₩4,900
- **메시지**: "더 나은 서버, 더 나은 코칭 추가 → 가격 조정"

**목표**:
- 가격탄성도 테스트 (이탈률 얼마나 증가?)
- 신규 획득 ARPU 상향
- MAU 50,000으로 확장 시 ₩7,900 검토

## 5. GTM Playbook (Go-to-Market)

### Phase A: 선행 콘텐츠 (3개월, MVP 빌드 前)

**목표**: Waitlist 500명 + X/Threads 팔로워 1,000명 + 마인드셰어 확보

**액션**:
- **주 2회 X 스레드 연재**: "쉼/DMN/집중력" 테마
  - 예: "코르티솔이 높으면 쉬는 것도 스트레스다", "뇌가 쉴 때 가장 창의적", "쇼츠 본 후 '뿌연 머리'는 도파민 고갈"
  - 톤: 과학-퍼스트, 마케팅 배제, 실무적 팁
  - 서명 스타일: "쉬어야 일이 잘 풀려 — MongCount" (매 스레드 마지막)

- **Waitlist 랜딩 페이지** (Vercel 배포):
  - 한 문장 가치 제안 + DMN 과학 gif + 3개 이점 + 이메일 입력 + "곧 만나요"
  - CTA: "사전 신청하기" → Google Form 수집

- **인플루언서 조사**: 생산성/웰니스 유튜버 (3명+) 대상 리스트 작성
  - 구체 채널: <U+200B>"달리는 INFP", "손에 잡히는 경제", "쌤씬" 등 번아웃/생산성 주제 채널

- **콘텐츠 개선**: Waitlist 피드백 기반 온보딩 카피 리파인

**결과물**:
- X 팔로워 1,000+ (₩0 광고)
- Waitlist 500+ 이메일

## 7. B2B2C Option (v2+)

### 가능성: 기업 웰니스 패키지

### 모델

**B2B2C = 기업(B2B) + 직원(C2C)**

```
MongCount (Platform)
    ↓
기업 HR팀 (구매자)
    ↓
직원 100명 (사용자, 무료/인센티브)
```

### 타겟 기업

| 업종 | 특징 | 예시 |
|------|------|------|
| **컨설팅/금융** | 스트레스 높음, 웰니스 예산 충분 | McKinsey, Goldman Sachs, 한국 빅3 카드사 |
| **테크** | 문화 중심, 복리후생 경쟁 | Kakao, Line, 국내 VC |
| **제약/의료** | 번아웃 높음 (의사/약사/간호사) | 메디컬 서비스 회사 |

### 가격 구조 (예시)

```
기본료: ₩50,000/월 (1-100명)
추가: 직원당 ₩1,000/월 (프리미엄 기능 자동 할당)

예: 50명 기업 = ₩50,000 + (50 × ₩1,000) = ₩100,000/월
=== MongCount 입장: C2C ₩145,000 vs B2B ₩100,000 (덜함) 하지만 안정성 있음
```

### 진입 조건 (Phase D 검토 기준)

- ✅ MAU 10,000+ (안정적 제품 증명)
- ✅ D30 리텐션 40%+ (중장기 가치 증명)
- ✅ HR팀 리드 5개 회사 (파일럿 가능성 검증)

### 위험

- ⚠️ 기업 정책 준수 (보안/개인정보): GDPR/PIPA 대응 비용
- ⚠️ SLA 약속: "알림 99% 신뢰성" 필요 (현재 iOS 70%는 부족)
- ⚠️ Sales 리소스: B2B는 C2C보다 영업 3배 필요

### 제외 이유 (MVP)

- Phase B 기간 2.5개월은 기업 영업 불가능
- 초기 제품 안정성이 B2B 수준 미달

## 10. Pricing Philosophy

### "가격은 신호다"

MongCount의 가격 책정 철학:

1. **너무 싸면 부담스럽다** (무료 심리 → 낮은 리텐션)
2. **너무 비싸면 못 나간다** (초기 사용자 장벽)
3. **적정가는 가치 수준을 신호한다**

### ₩2,900의 의미

- **일일 ₩96**: "커피 한 잔보다 저렴"한 심리
- **연 ₩34,800**: "한끼 외식"보다 저렴 → 구독 결정 쉬움
- **경쟁사 평균 ₩9,900 대비 70% 저렴**: "우리는 단순하고 싸다"는 신호

### 6개월 후 ₩4,900 인상

- "제품 개선 증명" (버그 수정, 새 기능, 안정성)
- "기존 사용자 존중" (Grandfathering)
- "신규 사용자 투자 수익화" (새 기능으로 정당화)
